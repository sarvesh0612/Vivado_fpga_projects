This project implements a **multi-operation Arithmetic Logic Unit (ALU)** using Verilog HDL.

## Description
The ALU performs various arithmetic and logical operations based on a control signal, similar to a processor datapath unit.

## Supported Operations
- Addition
- Subtraction
- AND, OR, XOR
- NOT
- Shift left / shift right

## Features
- Parameterized data width
- Combinational design
- Clean opcode-based operation selection

## Tools Used
- Verilog HDL
- Xilinx Vivado

## Applications
- CPU and microcontroller design
- Datapath design
- Digital system fundamentals
